# Sales-Data-DashBoard-Python-Base

A Python project where I analyzed and visualized sales data using pandas and matplotlib.

## What is this?

I made this project to get better at data analysis in Python. I didn't have a real dataset so I created one using numpy — 180 rows of sales data with products, regions and dates.

I wanted to see things like which product sells the most, which region is on top, and how sales look week by week.

## What I used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Charts included

- Total sales by product (bar chart with labels)
- Total sales by region
- Weekly sales trend (line chart)
- Region vs product heatmap

## How to run it

Just open the notebook in Google Colab and run all cells. It generates the dataset automatically so you don't need to download anything.

## Files

- `Sales-Data-DashBoard-Python-Base.ipynb` — the main notebook
- `sales_data.csv` — gets created when you run the notebook
- `sales_summary.csv` — summary exported at the end

## What I learned

Honestly pivot_table was confusing at first but once I understood it the heatmap became easy. Also learned how to add value labels on bar charts using get_height() which I didn't know before.
